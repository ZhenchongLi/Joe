# llms.txt Output Format Specification

## File Location
- **URL**: `https://fists.cc/llms.txt`
- **Local Path**: `public/llms.txt` (generated by Hugo)

## Content Structure

### Header Section
```
# Joe's Personal Website - LLM Content Index
# Generated: [ISO 8601 timestamp]
# Base URL: https://fists.cc/
# Language: zh-cn
# Description: Personal blog covering AI, products, algorithms, and thoughts

```

### Articles Section
```
# Articles (newest first)

[YYYY-MM-DD] [Category] [Title] - [URL]
[YYYY-MM-DD] [Category] [Title] - [URL]
...
```

### Example Output
```
# Joe's Personal Website - LLM Content Index
# Generated: 2025-09-30T10:30:00Z
# Base URL: https://fists.cc/
# Language: zh-cn
# Description: Personal blog covering AI, products, algorithms, and thoughts

# Articles (newest first)

2025-09-28 AI Multi-Agent Systems Analysis - https://fists.cc/posts/ai/multi-agent-analysis/
2025-09-25 AI Context Engineering Best Practices - https://fists.cc/posts/ai/context-engineering/

2025-09-15 Products Focus App Development - https://fists.cc/posts/products/focus/
2025-09-10 LeetCode Binary Tree Algorithms - https://fists.cc/posts/leetcode/binary-tree/
```

## Format Rules

### Line Format
- Comments start with `#`
- Article format: `[Date] [Category] [Title] - [URL]`
- Date format: `YYYY-MM-DD`
- URL format: Absolute URLs with https://

### Category Mapping
- "人工智能" → "AI"
- "thinking" → "Thinking"
- "products" → "Products"
- "leetcode" → "LeetCode"

### Character Encoding
- UTF-8 encoding
- Unix line endings (LF)
- ASCII-safe URLs (encoded if necessary)

### Size Constraints
- Maximum file size: 100KB
- Maximum line length: 200 characters
- Title truncation if needed: "Long Title... - URL"

## Error Handling

### Empty Content State
```
# Joe's Personal Website - LLM Content Index
# Generated: 2025-09-30T10:30:00Z
# Base URL: https://fists.cc/
# Language: zh-cn
# Description: Personal blog covering AI, products, algorithms, and thoughts

# No articles available
```

### Missing Metadata
- Missing category: "Uncategorized"
- Missing date: File modification date
- Missing title: Filename without extension
- Invalid characters: URL-encoded or replaced
